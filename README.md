# Focor Plugin Marketplace

Official Claude Cowork plugins by [Focor AI](https://focor.com).

## Available Plugins

| Plugin | Description |
|--------|-------------|
| **focor** | Cross-tool intelligence layer — morning briefings, context synthesis, and more |

## Install

Add this marketplace in Claude Cowork settings:

```
Focor/focor-cowork-plugin
```

Then install the `focor` plugin from the marketplace.
