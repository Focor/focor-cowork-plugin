# Focor — Claude Cowork Plugin

Cross-tool intelligence layer for Claude. Connects Slack, Notion, Linear, Calendar, Email and more. One-click install, full context across your workflow.

## v0.1 — Morning Briefing

This initial release includes:
- **`/morning-brief`** — Get a cross-tool synthesis for your day
- **`focor-intelligence` skill** — Teaches Claude when and how to use Focor

## Setup

1. Install the plugin from the Claude Cowork Plugin Marketplace
2. Connect your Focor account via OAuth
3. Type `/morning-brief` or ask Claude "what's going on today?"

## Requirements

- A Focor account with at least one connected tool (Slack, Gmail, Calendar, etc.)
